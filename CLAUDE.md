## Plugin Compatibility References

When changing plugin packaging, manifests, marketplace metadata, hooks, MCP config, commands, or runtime adapters, check the current docs for each supported target:

- Claude Code plugins: https://code.claude.com/docs/en/plugins
- Codex plugins: https://developers.openai.com/codex/plugins/build
- VS Code agent plugins: https://code.visualstudio.com/docs/agent-customization/agent-plugins

Adapter-specific files should document runtime packaging requirements without becoming the source of Forgekit's workflow behavior.
